# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1

- Initial `vscodenotes` extension scaffold.
- Added editor context menu command for creating local code notes.
- Added SQLite-backed note storage in VS Code global storage.
- Added hover provider for showing notes over marked code.
- Added command for removing notes at the cursor.
- Added cross-session database reload support.
- Added Prettier configuration for 4-space indentation.
